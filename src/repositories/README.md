# 📦 Repositories Layer (Data Provider)

이 폴더는 **데이터가 어디에 있든 상관없이** 앱에서 필요한 형식의 데이터를 제공하는 역할을 합니다.

### 📜 지켜야 할 규칙
1.  **데이터 추상화**: 상위 레이어(Actions)가 DB 문법이나 API 경로를 몰라도 되도록 숨기세요.
2.  **인프라 활용**: 기술적인 요청은 `infrastructure` 폴더의 클라이언트를 사용하여 수행하세요.
3.  **판단 금지**: "이 데이터가 중요한가?" 같은 비즈니스 판단은 하지 마세요. 그건 `domain`의 역할입니다. 여기서는 오직 데이터를 **안전하게 가져오고 저장**하는 것만 신경 씁니다.

# ⚙️ Infrastructure Layer (Technical Clients)

이 폴더는 최하위 계층으로, 외부 시스템과 대화하기 위한 **순수 기술 도구**들만 모아둡니다.

### 📜 지켜야 할 규칙
1.  **순수 기술**: 비즈니스 로직이나 도메인 지식을 절대 넣지 마세요. (예: `type=JSON` 같은 기술 옵션은 OK, "의료법 우선순위" 같은 도메인 옵션은 NO)
2.  **도구 역할**: DB 연결, HTTP 통신, AI 모델 호출 등 **"어떻게 접속하는가"**에 대한 구현만 담당합니다.
3.  **범용성**: 가능하면 이 프로젝트가 아닌 다른 곳에서도 바로 떼어다 쓸 수 있을 정도로 기술 중심적으로 작성하세요.
